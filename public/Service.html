Steps of creation Routing using express :

  • 1- install package.json using npm init -y
  • 2- install what you need in your project such as:
    • node.js : npm i node
    • express.js : npm i -D express
    • nodemon.js : npm i -D nodemon
    • semantic or bootstarp for designing
  • 3- create app.js file
  • 4- create Public folder for html files and Routes folder for routing files
